# Set-Up

Run `yarn` or `npm install`

No API is used

Create a `.env` file with `VITE_VEGA_API_KEY` providing your own key.
See `.env.example` for an example

## Running

`yarn run dev`

## Running tests

`yarn test:run`

or

`yarn test
